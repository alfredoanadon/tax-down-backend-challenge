## TaxDown Backend Challenge

# Realización de prueba tecnica de backend por Alfredo Anadon.

Consistia en desarrollar una API con Node.js y serverless como framework en el que se realizara una consulta
a una URL dada.

He realizado esta aplicación usando el puerto 3001 de mi localhost.

Esta API tiene 3 metodos GET. 

-El 1º coge los datos directamente del JSON.

-El 2º coge los datos de un archivo en raiz

-El 3º realiza ya la función de GET de la URL dada y la muestra en pantalla.

El despliegue de esta pequeña API se realiza con serverless y su archivo serverles.yml
